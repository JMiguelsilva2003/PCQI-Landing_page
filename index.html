SOBRE O PROJETO:

PLATAFORMA DE CONTROLO DE QUALIDADE INTELIGENTE

O PCQI é uma solução completa de IoT e Inteligência Artificial para automatizar a seleção de mangas. O sistema utiliza câmeras de alta resolução e redes neurais (CNN) para classificar cada fruta como "Madura" ou "Verde" em tempo real.

Funcionalidades Principais:

  • Classificação automática via Visão Computacional.
  • Dashboard Web para gestão de produção e estatísticas.
  • App Mobile para operadores com streaming de vídeo.
  • Hardware (Arduino) para separação física das frutas.
